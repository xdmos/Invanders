# Changelog

All notable changes to this project will be documented in this file.

## [2026.03.02] - 2026-03-15

- Refined the score overlay with an arcade cabinet frame and a cleaner leaderboard layout.
- Highlighted the top score with a tuned gold treatment that keeps the row readable.
- Added a wave-clear countdown overlay that automatically starts the next wave.

## [2026.03.01] - 2026-03-15

- Rebuilt the game from the default SpriteKit template into a full arcade-style Invanders experience.
- Added wave progression, scoring, high-score tracking, overlays, menus, and game-over flow.
- Added retro audio and haptic feedback generated directly in the app.
- Reworked the main game view setup for portrait play and improved runtime presentation settings.
- Added light, dark, and tinted app icon assets.
