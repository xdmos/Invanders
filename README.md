# Invanders

Invanders is a retro-inspired iOS arcade shooter built with SpriteKit.

## Features

- Portrait arcade gameplay designed for iPhone and iPad
- Wave progression with increasing enemy pressure
- Score tracking and persistent local high scores
- Menu, score, wave-clear, and game-over overlays
- Retro-style generated sound effects and haptic feedback
- Alternate light, dark, and tinted app icons

## Tech Stack

- Swift
- SpriteKit
- UIKit
- Xcode project for iOS

## Run Locally

1. Open `Invanders.xcodeproj` in Xcode.
2. Select the `Invanders` scheme.
3. Run the app on an iPhone or iPad simulator.
